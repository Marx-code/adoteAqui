# AdoteAQUI

**Visão Geral**:
- **Descrição**: Aplicativo mobile feito com Expo/React Native para cadastro e adoção de pets.

**Instalação**:
- **Clonar repositório**: `git clone <repo-url>`
- **Entrar na pasta do projeto**: `cd adoteAqui`
- **Instalar dependências**: `npm install` ou `yarn`

**Rodando o projeto**:
- **Iniciar servidor de desenvolvimento (Expo)**: `npm run start` ou `yarn start` — abre o Metro/Expo DevTools.
- **Abrir no Android (emulador ou dispositivo conectado)**: `npm run android` ou `yarn android`.
- **Abrir no iOS (macOS + Xcode)**: `npm run ios` ou `yarn ios`.
- **Executar no Web**: `npm run web` ou `yarn web`.

Após `npm run start` você pode escanear o QR code com o aplicativo Expo Go no seu dispositivo ou usar os botões do DevTools para abrir em emuladores.

**Estrutura principal do projeto**:
- **App.js**: ponto de entrada da aplicação.
- **index.js**: registro da aplicação.
- **package.json**: scripts e dependências.
- **src/screens/**: telas do app (Home, Login, Perfil, CadastroPet, DetalhePet).
- **src/components/**: componentes reutilizáveis.
- **src/assets/**: imagens e recursos estáticos.
